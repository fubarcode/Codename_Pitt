# Spec Kit Scaffolding

This directory reserves space for Spec Kit / speckit workflow assets such as project memory, templates, and scripts.

## Layout

```text
.specify/
  memory/
    constitution.md
  templates/
```

## Workflow

1. Update or review `.specify/memory/constitution.md`.
2. Write the feature specification in `specs/<feature>/spec.md`.
3. Produce the implementation plan in `specs/<feature>/plan.md`.
4. Derive executable tasks in `specs/<feature>/tasks.md`.
5. Implement only after the spec, plan, and tasks are reviewed.
